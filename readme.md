# fr

a cli tool to recursively find repositories in your directories

usage:

```sh
# list repos in a dir
fr [dir]
```

example output:

```sh
$ fr ~/.local/share/nvim 
lazy/lazy.nvim
lazy/LuaSnip
lazy/cmp-nvim-lsp
lazy/cmp-buffer
lazy/cmp_luasnip
lazy/fzf-lua
lazy/nvim-dap-ui
lazy/nvim-lspconfig
lazy/nvim-surround
lazy/nvim-nio
lazy/nvim-web-devicons
lazy/nvim-treesitter
```
